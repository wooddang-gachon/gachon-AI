# 🐚 전복(Abalone) 나이 예측 딥러닝 프로젝트

본 프로젝트는 전복의 성별, 크기, 무게 등 신체 측정 데이터를 활용하여 전복의 나이를 나타내는 **'껍질의 고리 수(Rings)'**를 예측하는 딥러닝 모델을 구현한 사례입니다.

---

## 📂 프로젝트 구성
- **AbaloneDLClassification.ipynb**: 전복의 고리 수를 특정 '클래스'로 구분하여 분류하는 모델 (다중 분류)
- **AbaloneDLRegression.ipynb**: 전복의 고리 수를 연속적인 '수치'로 직접 예측하는 모델 (회귀)
- **abalone.csv**: 전복 데이터셋 (성별, 길이, 지름, 높이, 전체 무게, 껍질 무게 등)

---

## 🛠 데이터 전처리 (Data Preprocessing)

1. **라벨 인코딩 (Label Encoding)**:
   - 문자열 데이터인 `Sex`('M', 'F', 'I')를 모델이 이해할 수 있는 숫자(0, 1, 2)로 변환합니다.
   - 주의: 'I'(Infant)를 누락할 경우 데이터에 결측치(NaN)가 발생하여 학습 시 `loss: nan` 오류가 생길 수 있습니다.

2. **피처 스케일링 (Feature Scaling)**:
   - `StandardScaler`를 사용하여 모든 수치 데이터의 평균을 0, 표준편차를 1로 맞춥니다.
   - 단위가 서로 다른 특징들(무게와 길이 등)의 범위를 맞추어 학습 속도와 성능을 향상시킵니다.

---

## 🧠 딥러닝 모델 상세 설명

### 1. Sequential 모델 (설계 방식)
`Sequential`은 케라스(Keras)에서 레이어를 **순차적으로 쌓는 방식**입니다. 데이터가 입력층에서 출력층까지 한 방향으로 흐르는 가장 직관적인 구조입니다.

### 2. 레이어(Layer) 설정
- **Dense(노드 수)**: 각 층에 있는 계산 유닛의 개수입니다.
- **Activation("relu")**: 활성화 함수로, 0보다 큰 값만 통과시키는 필터 역할을 하여 복잡한 관계를 학습하게 돕습니다.
- **Dropout(0.3)**: 학습 시 일부 뉴런을 무작위로 꺼서 특정 데이터에만 과하게 익숙해지는 **과적합(Overfitting)**을 방지합니다.

### 3. 학습 설정 (Compile & Fit)
- **Optimizer("adam")**: 오답을 정답으로 수정해 나가는 최적화 도구(내비게이션)입니다.
- **Loss (손실 함수)**: 모델이 얼마나 틀렸는지 계산하는 공식입니다.
  - 분류: `categorical_crossentropy` (One-hot 필요)
  - 회귀: `mse` (Mean Squared Error)
- **Epochs(50)**: 전체 데이터를 50번 반복해서 학습하는 횟수입니다.
- **Batch size(16/32)**: 데이터를 한 번에 몇 개씩 묶어서 공부할 것인지 결정하는 단위입니다.

---

## ⚠️ 트러블슈팅 (Troubleshooting)

1. **Loss가 `nan`으로 나오는 경우**:
   - 데이터 전처리 중 결측치(NaN)가 발생했는지 확인하십시오. (예: `Sex` 매핑 시 'I' 누락)
2. **ModuleNotFoundError (Sklearn)**:
   - 라이브러리 임포트 시 철자 오류(`pregprocessing` -> `preprocessing`)를 확인하십시오.
3. **회귀 모델의 정확도가 낮은 경우**:
   - 타겟 데이터(`y`)를 원-핫 인코딩했는지 확인하십시오. 회귀 모델은 인코딩 없이 실제 수치 데이터를 그대로 사용해야 합니다.
